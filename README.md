## Features
1. Scrolling through popular and top rated of movies and tv series from [TMDB API](https://developer.themoviedb.org/reference/getting-started) including now playing movies and airing today tv series
2. Search movies and tv series that available in [TMDB API](https://developer.themoviedb.org/reference/getting-started)
3. See the details of selected movie or tv series
4. Add watchlist of selected movie and tv series

## Build App Setup Recomendation
- Flutter 3.38.6
- Dart 3.10.7
- JBR 21

## Code Design & Structure
This project is using BLoC State Management and Clean Architecture Pattern
